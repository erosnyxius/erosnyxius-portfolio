# Portfolio

A modern, responsive portfolio website built with Next.js and Tailwind CSS.

## Features

- **Single Page Application** - Smooth scrolling between sections
- **Dark Theme** - Elegant black and gray color scheme
- **Responsive Design** - Works on all devices
- **Animated Elements** - Floating cards and twinkling stars
- **Modern UI Components** - Cards, badges, buttons, and forms

## Sections

- **About Me** - Personal introduction and background
- **Experience** - Professional work history
- **Projects** - Showcase of featured work
- **Strategy & Services** - Services offered
- **Tech Stacks** - Technologies and skills
- **Education** - Academic background and certifications
- **Contact** - Contact form and information

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- **Framework**: Next.js 14
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Components**: Custom UI components with class-variance-authority

## Customization

Edit the component files in `/components` to customize content. Update `/app/globals.css` to modify theme colors.
